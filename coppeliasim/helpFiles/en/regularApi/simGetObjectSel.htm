Regular API function

simGetObjectSel / sim.getObjectSel

Description Retrieves the handles of selected objects. See also sim.setObjectSel
C/C++
synopsis 		int* simGetObjectSel(int* cnt)
C/C++
parameters
cnt: the number of selected objects
C/C++
return value
a pointer to the selected objects. The user is in charge of releasing the buffer via simReleaseBuffer
Lua
synopsis 		int[] selectedObjects=sim.getObjectSel()
Lua
parameters
Lua
return values
selectedObjects: the handles of selected objects
Python
synopsis		 list selectedObjects=sim.getObjectSel()
